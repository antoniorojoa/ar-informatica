# AR Informática

![AR Informática Logo](assets/img/logo.png)

Este repositorio contiene el código fuente para el portal web de **AR Informática**, un servicio técnico de reparación de ordenadores (PC) y dispositivos móviles (smartphones) situado en **Zahara de la Sierra**.

## 🚀 Sobre el proyecto

La web está diseñada para un **emprendedor individual** que ofrece servicios de reparación y puesta a punto de tecnología. El sitio destaca por su diseño moderno, rápido y responsive.

### Características Principales

*   **Diseño Moderno & Premium:** Tema oscuro con efectos de _glassmorphism_, desenfoques, y esquemas de color vibrantes (azul eléctrico y naranja).
*   **Totalmente Responsive:** Adaptable a todo tipo de pantallas (móviles, tablets y monitores de escritorio).
*   **Animaciones Dinámicas:** Animaciones al hacer scroll usando AOS (Animate On Scroll), efectos de tilt 3D en las tarjetas, y partículas animadas en el hero section.
*   **Formulario de Contacto:** Integración directa para abrir el cliente de correo del usuario usando `mailto:`.

## 🛠️ Tecnologías Utilizadas

*   **HTML5**
*   **CSS3** (con variables CSS y animaciones personalizadas)
*   **JavaScript Vanilla**
*   **Bootstrap 5.3** (Framework CSS)
*   **Bootstrap Icons**
*   **AOS** (Librería de animaciones)

## 🌐 Secciones del Sitio

1.  **Hero:** Sección de inicio atractiva con propuesta de valor y acceso rápido a contacto.
2.  **Servicios:** Catálogo de servicios (Formateo, Reparación de PC/Móviles, Puesta a punto, Recuperación de datos, etc.).
3.  **Sobre Mí:** Información personal sobre la pasión y dedicación del emprendedor detrás de AR Informática.
4.  **Proceso:** Cómo funciona el servicio paso a paso.
5.  **Ventajas:** Razones para elegir el servicio (agilidad, precios honestos, garantía y trato personal).
6.  **Contacto:** Formulario de envío rápido de dudas y ubicación en Zahara de la Sierra.
